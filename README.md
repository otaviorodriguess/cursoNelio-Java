# 📚 Curso de Java - Nélio Alves

Este repositório contém minha jornada de aprendizado em Java, baseada no curso do professor Nélio Alves.

## 🚀 Sobre

Aqui estou praticando conceitos fundamentais da linguagem Java através de exercícios e pequenos programas.

O foco é evoluir na lógica de programação e na Programação Orientada a Objetos (POO).
